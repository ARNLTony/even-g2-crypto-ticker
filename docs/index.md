# Crypto Ticker for Even G2

An Even Hub web app that streams live cryptocurrency prices and charts to the Even Realities G2 smart glasses.

- **Source code & docs:** [github.com/ARNLTony/even-g2-crypto-ticker](https://github.com/ARNLTony/even-g2-crypto-ticker)
- **Privacy policy:** [PRIVACY](PRIVACY)
- **License:** [MIT](https://github.com/ARNLTony/even-g2-crypto-ticker/blob/master/LICENSE)

Data is fetched from public [Binance](https://www.binance.com/) APIs only. No personal information is collected.
